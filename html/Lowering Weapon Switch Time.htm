Lowering Weapon Switch Time

This tutorial covers the code changes needed to lower the time it takes to swap weapons.
Thanks to Dale for the idea for this tutorial.

Files we will change.
game/bg_pmove.c

KEY:
Indicates changed lines

Step 1: This is really simple change so lets get started. Open up bg_pmove.c and goto line 1371 (or there abouts), it should look like this.

pm->ps->weaponTime += 200;

Step 2: Change the value of 200 to 0, save an compile it. Thats all there is to it :)

Enjoy, I hope this helps you all out.
